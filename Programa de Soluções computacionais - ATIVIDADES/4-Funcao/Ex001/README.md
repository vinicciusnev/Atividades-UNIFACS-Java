## Atividade 01 da Lista de Funções

Sejam bem vindos ao projeto.

## Enunciado

aça um programa para imprimir:
    1
    2   2
    3   3   3
    .....
    n   n   n   n   n   n  ... n
para um n informado pelo usuário. Use uma função que receba um valor n inteiro e imprima até a n-ésima linha.
